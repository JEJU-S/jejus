## 역할

|권내영|박병준|오정환|최재원
|-----|-----|-----|-----|
|- 웹 프로토타입 제작, 서비스 로직, 세부기능 구성</br></br>Router, Middleware, session, callback 구현 등 Node.Js기반 서버 구축 </br></br>- Google OAuth2.0 API를 통한 Google Login 구현 및 PaaS-Ta server 접근 권한 부여</br></br>- Kakao local API를 활용한 실시간 키워드 검색, NAVER dynamic map을 활용한 실시간 map, 실시간 채팅, 칸반보드 등 socket.io 기반 실시간 서비스 페이지 구축</br></br>- ejs, js 사용해 main page 제외한 전체 페이지의 전반적인 기능 구현 |- 기획단계에서 시장 분석 및 요구사항 분석을 통한 서비스 로직 작성</br></br>-MongoDB local test server 구축</br></br>- Node.Js Mongoose module을 활용하여 binding된 MongoDB server와 통신 구축</br></br>- MongoDB server에 회원 정보, 여행 일정 DB 구축 (Google Login을 통한 회원가입 구현, 여행 일정 삭제 구현)|기획단계에서 시장 분석, 경쟁사 분석을 통한 서비스 세부기능 정의</br></br>- Photoshop을 사용하여 프로젝트 웹사이트 프로토타입 제작</br></br>- HTML, CSS, Javascript, JQuery Library를 사용하여 - UI/UX 디자인 (초기화면, 프로젝트 프로필 페이지, 프로필 수정 페이지, 새 계획 생성 페이지, 내 그룹 보기 페이지, 초대 기능 디자인, 전체일정 보기 기능 디자인, 세부 일정 보기 기능 디자인,</br></br>- 여행 일정 편집페이지, 여행 일정 편집페이지 권역 선택 기능 및 디자인 여행 일정 편집페이지 도움말 기능 디자인)|개방형 클라우드 플랫폼(PaaS-Ta Playpark)에 Node.Js 앱 배포 후 MongoDB binding 및 배포된 애플리케이션 환경설정</br>Node.Js Mongoose module을 활용하여 binding 된 MongoDB server와 통신 구축</br></br>- MongoDB server에 회원 정보, 여행 일정, 추천 여행지 DB 구축 (여행 일정 생성 구현, 초대장 발송 및 수락/거절 구현, 여행 일정 확인 및 편집페이지 내 socket - DB 통신 구현)</br></br>- 데이터 수집 및 정규화를 통한 추천모델을 생성하고 제주도 관광지 추천 데이터 셋 구축|
